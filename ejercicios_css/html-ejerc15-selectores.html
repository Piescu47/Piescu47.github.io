Tipos de selectores que entran en examen

  1. Clase (class=””)

    2. Es un identificador de diferentes elementos dándoles un tipo de clase concreto. Se utiliza para facilitar la modicicación de todos los pertenecientes a una misma clase mediante CSS.

  2. Identificador (id=””)

    3. Es un tipo de selector cuyo propçosito es vincular a un elemento con un nombre concreto para modificarlo con el CSS.

    id="er"
    #er { color: blue; }

  3. Todos los elementos (*)

    4. Es un tipo de identificador que selecciona todo el documento para que este sea modificado mediante CSS.

    * { text-align: center }

  4. Varios elementos ( p, em, h1, h2)

    5. Estos grupos de selectores harán que todos los elementos que pertenezcan a esas etiquetas se vean modificados utilizando lenguaje de CSS

    h1 { text-align: center; color: red; } h2 { text-align: center; color: red; } p { text-align: center; color: red; }

  5. Elementos contenidos en otros elementos (ol li)

    6. Son etiquetas tuilizadas para crear diferentes listas de elementos, las ordenadas (ol) y las que no tienen orden (ul). Al utilizarlas en el lenguaje CSS, se modificarám todos los elementos pertenecientes a estas listas.

  6. Sobrevuleo (p:hover)

    7. Es una interacción en la interfaz del usiario que se utiliza para darle un segundo estado a cualquier elemento, cuándo un usuario interactúa con este, por ejemplo, al pasar el cursor por encima.

    a:hover { color: orange; }
